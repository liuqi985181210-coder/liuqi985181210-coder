# 本地数据与办公自动化

这里公开的是可复现、只使用虚构数据的实现样例。当前主要处理固定的 Excel / CSV 重复流程：多表合并、按字段拆分、对账汇总、数据同步、异常隔离和重试。

## 当前可承接

- `.xlsx` / `.csv` 多文件按表头合并；
- 按门店、客户、部门等固定字段拆分；
- 从固定明细生成对账汇总、客户明细或月报；
- CSV / API 事件的去重、状态同步、失败重试和异常清单；
- 先用 1–2 份脱敏样例确认输入、规则、输出和验收结果。

基础小单对应一个能单独验收的结果；需要反复使用的 Windows 本地工具从 199 元起。范围和费用先书面确认，不以低价入口承诺无限修改。

## 不承接

- 未经授权的数据采集、账号托管、验证码绕过、刷量或群发；
- 银行、病历、身份证等高风险数据的远程托管；
- “所有 Excel / PDF 都能处理”或“100% 准确”的无限范围；
- 真伪、税务、法律或报销审批结论。
